# Building the AI Bank — Book Landing Page

A fully responsive, SEO-optimised landing page for **Building the AI Bank: Speed, Judgment and Power** by Emmanuel Daniel.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main HTML structure, all sections, Schema.org metadata, Open Graph tags |
| `style.css` | All styling — warm `#faf7f3` background, black/grey/white palette, responsive layout |
| `main.js` | Tab switching, scroll behaviour, newsletter form handler |
| `book-cover.png` | Book cover image (replace with high-res version if needed) |

## Sections

1. **Hero** — Book cover, title, excerpt, CTA buttons
2. **About the Author** — Emmanuel Daniel biography and credentials
3. **Table of Contents** — 4-tab layout (Part I, II, III, Forewords)
4. **Key Ideas** — 9 concept cards
5. **Reviews** — 5 expert endorsements + rating
6. **Who Must Buy This Book** — 6 audience profiles
7. **Blogs, Videos & Articles** — 5-tab content feed (Geopolitics, Finance, Travel, Reviews, Footnotes)
8. **Get Your Copy Today** — 8 retail links (Amazon, Google Books, B&N, Amazon UK, Book Depository, Waterstones, World Scientific, JD.com)
9. **Newsletter** — Email subscription form

## Hosting on GitHub Pages

1. Push all files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your site will be live at `https://<username>.github.io/<repo>/`

## Fonts

Uses Google Fonts (loaded from CDN in `index.html`):
- **Syne** — headings and display text
- **Instrument Serif** — pull quotes and body serif
- **DM Sans** — UI text and body copy

No build step required. Plain HTML/CSS/JS — deploy anywhere.
